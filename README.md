### This repo is a demo for testing Github Actions
```
[main]
 ┣━ .github
 ┃   ┗━ workflows
 ┃       ┗━ python-tests.yml       # GitHub Actions 설정 파일
 ┣━ src
 ┃   ┣━ __init__.py                # 패키지 임포트를 위한 파일(비어있음)
 ┃   ┗━ factorial.py               # 팩토리얼 로직 구현
 ┣━ tests
 ┃   ┗━ test_factorial.py          # 테스트 코드
 ┣━ requirements.txt               # 필요한 라이브러리 명시
 ┗━ README.md                      # 프로젝트 사용법, 설명
```
